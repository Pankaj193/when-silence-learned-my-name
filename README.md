---

## 📖 About

This is a **story-driven web experience** designed to feel like a forgotten newspaper rather than a traditional website.

The interface unfolds through scrolling — where the **cover slowly reveals the first chapter**, blending design, motion, and narrative into a single continuous experience.

---

## ✨ Features

* 📰 Vintage **newspaper-style layout**
* 🌅 **Sunlight & shadow aesthetics** for mood
* 🎬 **Scroll-based animations** (headline reveal → article transition)
* 📖 Seamless **cover-to-chapter transformation**
* 🖤 Minimal, immersive storytelling

---

## 🚀 How to Run

```bash
npm install
npm run dev
```

Then open the local URL in your browser.
